# Ants Race Challenge

Challenge for Farfetch.

## Demo

![](demo-ios.gif)

## Run Locally

Clone the project

```bash
  git clone https://github.com/Eduardo-Queiroz/ants-race.git
```

Install dependencies

```bash
  npm install
```

IOS extra step

```bash
  cd ios
  pod install
```

IOS

```bash
  yarn ios
```
